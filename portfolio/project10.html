Vue-based Typescript website

Photo gallery from Unsplash API

This is my first website using Vue.js, and also incorporates Typescript. The website displays a grid of photo cards. The data used to populate this grid is fetched on mount from Unsplash API. In this case, the request is for 30 random photos (the maximum the API allows for a single request) with portrait orientation. The photos are organized in descending order of popularity (measured by number of views) using JavaSript's sort method.

Once we parse the JSON data from the API we are left with an array of 30 objects. We map the array using vue's built in "v-for" directive. For each object we create a "PhotoCard" component instance and we pass the required props for each instance. The data thas is dynamically produced with props on each instance include: the src link for the image, the name of the photographer (and its associated link to their Unsplash profile), number of views, among others.

The grid of photos is styled with CSS grid and is fully responsive. Each image becomes a little bit transparent on hover and the cursor shifts to pointer to indicate interactivity. Clicking on it takes us to a larger version of the photo in a new tab.
